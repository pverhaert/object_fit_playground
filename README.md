# CSS Object-Fit Visualizer

A simple interactive tool to demonstrate how CSS `object-fit` and `object-position` properties work.

## How to Use
1. Clone or download this repository.
2. Open `index.html` in your web browser.
3. Use the control panel to:
   - Switch between `fill`, `contain`, `cover`, and `none` modes.
   - Adjust the `object-position` using the X and Y sliders.
   - Reset to defaults using the "Reset Defaults" button.

## Hosting on Netlify
You can easily host this project for free on Netlify.

### Option 1: Drag and Drop (Manual)
1. Go to [app.netlify.com](https://app.netlify.com) and log in.
2. Go to the "Sites" tab.
3. Drag the project folder (containing `index.html`) onto the "Drag and drop your site output folder here" area.
4. Your site will be live instantly!

### Option 2: Git Integration
1. Push this code to a GitHub/GitLab/Bitbucket repository.
2. Log in to Netlify and click "Add new site" > "Import an existing project".
3. Connect your Git provider and select this repository.
4. Click "Deploy Site" (no build command or publish directory needed for this static site).
